# Weather App

도시별 시간대별 날씨와 아이콘, 그리고 현재 날씨에 어울리는 배경화면을 보여주는 React 기반 날씨 웹앱입니다.

## 주요 기능
- 도시 검색 및 선택
- 3시간 간격의 시간대별 온도/날씨/아이콘 표시
- 현재 날씨에 어울리는 배경화면 자동 적용

## 실행 방법
1. 패키지 설치
   ```bash
   npm install
   ```
2. 개발 서버 실행
   ```bash
   npm start
   ```
